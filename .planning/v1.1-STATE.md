## Context

### What This Is

v1.1 builds on v1 by adding YouTube recipe import capability. Users can paste a YouTube video URL and have the recipe automatically populated from the video metadata (title, description, thumbnail) with NLP parsing of ingredients and instructions.

### Tech Stack (inherited from v1)

- Django 6.0.3
- PostgreSQL 14+
- Tailwind CSS 4.x + DaisyUI 5.x
- Alpine.js for selective enhancement
- YouTube Data API for video metadata

| Decision | Rationale | Status |
