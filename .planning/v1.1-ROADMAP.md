## Phases

- [ ] **Phase 1: URL Import & Validation** - User can paste YouTube URL and initiate import
- [ ] **Phase 2: Metadata Fetch** - Video metadata is retrieved from YouTube

- [ ] **Phase 4: Form Population & Photo** - Recipe form populated with imported data

---

## Phase Details

### Phase 1: URL Import & Validation
**Goal**: Users can paste a YouTube URL into a field and initiate import  
**Depends on**: Nothing (first phase)  
**Requirements**: IMP-01, IMP-02, IMP-03, IMP-04  
**Success Criteria** (what must be TRUE):
  1. User can enter a YouTube URL in a dedicated import field
  2. System validates URL format (youtube.com or youtu.be) before processing
  3. Import button shows loading state while fetching is in progress
  4. Invalid URL shows clear, user-friendly error message
**Plans**: TBD

### Phase 2: Metadata Fetch
**Goal**: Video metadata is retrieved from YouTube API and prepared for parsing  
**Depends on**: Phase 1  
**Requirements**: IMP-05, IMP-06, IMP-07  
**Success Criteria** (what must be TRUE):
  1. Video title is fetched and displayed in recipe form title field
  2. Video description is fetched and used as raw content for parsing
  3. Thumbnail URL is captured and available for recipe photo
**Plans**: TBD

### Phase 3: Content Parsing
**Goal**: Ingredients and instructions are parsed from video description using NLP  
**Depends on**: Phase 2  
**Requirements**: IMP-08, IMP-09, IMP-10, IMP-11, IMP-12, IMP-13, IMP-14, IMP-15  
**Success Criteria** (what must be TRUE):
  1. Ingredients are parsed from description using NLP/entity recognition
  2. Parsed ingredients appear as pre-filled rows in ingredient form
  3. Instructions/steps are parsed from description text or timestamps
  4. Parsed instructions appear as pre-filled rows in instruction form
  5. User can edit or delete any parsed ingredient before saving
  6. User can edit or delete any parsed instruction before saving
  7. User can add additional steps manually
  8. Unparseable content is noted for user to add manually
**Plans**: TBD
**UI hint**: yes

### Phase 4: Form Population & Photo
**Goal**: Recipe form is populated with all imported data, user can modify and save  
**Depends on**: Phase 3  
**Requirements**: IMP-16, IMP-17, IMP-18, IMP-19, IMP-20, IMP-21  
**Success Criteria** (what must be TRUE):
  1. Recipe form opens pre-filled with title, ingredients, instructions, and photo
  2. User can modify any field before saving recipe
  3. User can upload a new photo if no thumbnail was imported
  4. YouTube thumbnail appears as default recipe photo
  5. User can replace thumbnail with their own uploaded photo
  6. User can remove photo entirely if desired
**Plans**: TBD
**UI hint**: yes
